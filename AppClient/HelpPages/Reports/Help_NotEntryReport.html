Activity Not Entry Report

REPORT DESCRIPTION

The report will display list of subordinate of the logged in user (if they are present on that day, i.e., users who have checked in for a day) who did not entered any activity during the specified date range. The report helps user to know the list of days his subordinates did not entered activities.

ACCESSIBLE BY

End User: Can view dates on which no activities were entered by the logged in user.

Supervisor or Manager: Can view the dates for which activities were not entered by the logged in user and subordinates attached to him / her.

Administrator: Can view all users in all locations, who did not enter any activity on a day.

STEPS TO ACCESS THE REPORT

  • From the Home screen click on Reports
  • From the Reports screen click on Activity Not Entry Report

STEPS TO RUN THE REPORT

  • Enter the From Date and To Date within which we wish to check for the activities not entered instances.
  • Select the Location for which we need to view the users who have not entered any activity.
  • Select any supervisors (users with subordinates).
  • If any supervisor or manager is attached to the user. The user can select the supervisor or manager to view details of team members attached to that particular supervisor or manager
  • Click on View button to run the report
  • To exit without running the report click the Cancel button

OUTPUT

  • After inputting the date range, click on the View button to view the report
  • The report will display the list of users with Emp ID, Role and the Date on which activities were not entered by them.

Output Fields

The below fields are displayed in the report

  • Employee ID, First Name, Last Name, Date (Not Entered Date), Role Name and Email ID (User's Email ID), Check In/Out Duration.

REPORT OPTIONS

  • The left arrow and right arrow button helps in navigating to the previous page and next page of the report.
  • The forward and backward button helps to navigate to last page and first page of the report.
  • We can also search desired value by entering the value in the search box and click on Find button. Click Next to search the value again.
  • Click on the Save icon, which provides option to export the report in Excel, Word or PDF.
