Client Management

FUNCTION

  • The Clients page helps us to add a new client to the master or modify an existing client. This page helps us to create clients (e.g., Atari, Namco Badai) and assign managers for these clients

ACCESSIBLE BY

  • Users who have administrator access.

STEPS TO CREATE A NEW USER

  • From the Home page, Admin  Clients
  • This will take us to the Clients Management page.

  • Click on the Add New button to add a new client
  • In the pop-up window enter the name of the client.
  • Enter a brief description about the client if needed

Steps to select a manager

  • Press F2 or click on the icon to open the search window
  • Select the City or enter Name or Role or Email ID of the manager to search.
  • Click on the Filter button to display employees meeting the above search criteria.
  • To select a manager click on the Select button or click on the Clear button to clear the search criteria and results.
  • From the Add Client pop-up window click on the Add button to add the client to the masters or click on Cancel button to exit without saving changes.

STEPS TO MODIFY EXISTING CLIENT

  • From the Client Management page enter few characters of the name of the client that has to modified and click on the filter button to search the client.
  • Selecting active or inactive in the status field will restrict the search to active clients and inactive clients respectively.
  • To clear the current search criteria, search results and enter new criteria click on the clear button.
  • Click on the Edit button against the name of client that has to be modified.

  • Edit Client page has similar fields and functions like Add Client page. Apart from this the Edit Client page has Active check box. To make a client inactive uncheck the check box.
  • Make desired changes in the Name or description field or we can change the manager to whom the client is assigned. View the Steps to select a manager to select a different manager.
  • Give the reason for modification in the reason box.
  • Click on the Update button to save the changes and close the pop-up window or click on Cancel button to exit the pop-up window without saving changes.
