Activity Reset View

FUNCTION

  • The Activity Reset page helps us to reset any approved activities, if any activity was approved by mistake.
  • User will get list of activities approved by them.

ACCESSIBLE BY

  • Users with administrator access, supervisor access or Manager access.

STEPS TO RESET AN APPROVED ACTIVITY

  • To enter the Employee Name, press down arrow key to list the employees whose activity were approved by this user or type few characters of employee name to display list matching the characters. Select the employee from the drop down menu
  • Then select the Activity Entered From Date and Activity Entered To Date.
  • Click on the Search button to list activities matching the search criteria.
  • Click on the Clear button to clear the search criteria and search results.
  • Select the activity that has to reset by marking the check box on the left side of the activity.
  • Give the reason for resetting the activity
  • Click on the Reset button to reset the selected activities or click on the clear button to exit without rejecting the activity
