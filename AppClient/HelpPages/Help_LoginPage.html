LOGIN PAGE

FUNCTION

The login page enables a user / employee to access the application. User name can only be created by a user / employee having administrator access.

To access the application

  • Open a compatible web browser and enter the web address http://tt.e4e.com and press enter.
  • Enter the assigned user name, password
  • Then click on the Login button to access the application.

To reset forgotten password

  • In the same login page, enter the user name and click on the “Forgot Password” button.
  • You will receive an email link to the email ID which was already entered into the application when creating the user.
  • Click on the link or copy and paste the link in any of the compatible web browser and press Enter, which will take us to the password reset page.
  • Enter your user name and new password to reset the password.

Accessible by

  • All users with a valid user name and password.

Note

User can access the application only from specified locations. If a user needs to access the application from any locaiton, the user needs to get the access from superior
