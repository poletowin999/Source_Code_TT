Location Management

FUNCTION

  • The Location page helps us to add a new Location type to the master or modify an existing Location.

ACCESSIBLE BY

  • Users who have administrator access.

STEPS TO ADD A LOCATION

  • From the Home page, Admin  Location
  • This will take us to the Location Management page.

  • Click on the Add New button to add a new Location
  • In the pop-up window enter the City, State, Country.
  • Select the Time zone, by entering few characters or press Down arrow Key to get the masters and select the Time zone from the drop down.
  • Enter a brief description about the Location if needed
  • Click on the Add button to add the Location to the masters or click on Cancel button to exit without saving changes

STEPS TO MODIFY EXISTING LOCATION

  • From the Location Management page, enter few characters of the name of the City, State, Country and click on the filter button to search any location.
  • Selecting active or inactive in the status field will restrict the search to active Location and inactive Location respectively.
  • To clear the current search criteria, search results and enter new criteria click on the clear button.
  • Click on the Edit button against the name of Location that has to be modified. This will take us to the Edit Location page.

  • Edit Location page has similar fields and functions like Add Location page. Apart from this the Edit Location page has Active check box.
  • Make desired changes in the City, State, County or Time Zone.
  • Give the reason for modification in the reason box.
  • Click on the Update button to save the changes and close the pop-up window or click on Cancel button to exit the pop-up window without saving changes.
