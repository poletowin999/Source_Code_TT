WorkType Management

FUNCTION

  • The Work Type page helps us to add a new Work type to the master or modify an existing work type.
  • Work type is basically a classification of activity. Like Test Hours, Meeting, Leave, Compliance Test hours, etc.,

ACCESSIBLE BY

  • Users who have administrator access.

STEPS TO ADD A WORK TYPE

  • From the Home page, Admin  Work Type
  • This will take us to the Work Type Management page.

  • Click on the Add New button to add a new Work Type
  • In the pop-up window enter the name of the Work Type.
  • Enter a brief description about the Work Type if needed
  • Select whether the type of the activity from the Activity Type drop down menu. This enable us to classify whether the work type is project related, miscellaneous, etc.,
  • Click on the Add button to add the work type to the masters or click on Cancel button to exit without saving changes

STEPS TO MODIFY EXISTING WORK TYPE

  • From the Work Type Management page, enter few characters of the name of the Work Type that has to be modified and click on the filter button to search the Work Type.
  • Selecting active or inactive in the status field will restrict the search to active work type and inactive work type respectively.
  • To clear the current search criteria, search results and enter new criteria click on the clear button.
  • Click on the Edit button against the name of Work Type that has to be modified.

  • Edit Work Type page has similar fields and functions like Add Work Type page. Apart from this the Edit Work Type page has Active check box. To make a Work Type inactive uncheck the check box.
  • Make desired changes in the Name or Description or Activity Type.
  • The Consider For Report shows whether the work type is included for reporting purpose or not. If the check box is checked the work type will be included in all reports. This field is not editable
  • Give the reason for modification in the reason box.
  • Click on the Update button to save the changes and close the pop-up window or click on Cancel button to exit the pop-up window without saving changes.
