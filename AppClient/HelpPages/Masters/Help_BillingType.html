Billing Types Management

FUNCTION

  • The Billing Type Page helps us to add a new Billing type to the master or modify an existing Billing type.
  • The billing type is used in payroll; salary calculation differs based Regular or Overtime bill type

ACCESSIBLE BY

  • Users who have administrator access.

STEPS TO ADD A BILLING TYPE

  • From the Home page, Admin  Billing Type
  • This will take us to the Billing Type Management page.

  • Click on the Add New button to add a new Billing Type
  • In the pop-up window enter the name of the Billing Type.
  • Enter a brief description about the Billing Type if needed
  • Click on the Add button to add the Billing Type to the masters or click on Cancel button to exit without saving changes

STEPS TO MODIFY EXISTING BILLING TYPE

  • From the Billing Types Management page, enter few characters of the name of the Billing type that has to modified and click on the filter button to search the billing type.
  • Selecting active or inactive in the status field will restrict the search to active billing type and inactive billing type respectively.
  • To clear the current search criteria, search results and enter new criteria click on the Clear button.
  • Click on the Edit button against the name of Test that has to be modified.
  • Click on the Edit button against the name of billing type that has to be modified. This will take us to the Edit Billing Type pop-up window.

  • The Edit Billing Type pop-up window has similar fields and functions like Add Billing Type page. Apart from this the Edit Billing Type page has Active check box. To make a Test inactive uncheck the check box.
  • Make desired changes in the Name or description field
  • Give the reason for modification in the reason box.
  • Click on the Update button to save the changes and close the pop-up window or click Cancel button to exit without saving changes.
