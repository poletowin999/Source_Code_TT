IP Config

FUNCTION

  • The IP Config page helps us to give access to IP address through which the application can be accessed.

ACCESSIBLE BY

  • Users who have administrator access.

STEPS TO ADD AN IP Address

  • From the Home page, Admin  IP Config
  • This will take us to the IP Config page.

  • Click on the Add New button to add a new IP Address
  • Select the link type
  • Enter the IP Address through which the application will be accessed
  • Enter the Location

STEPS TO MODIFY EXISTING IP

  • From the IP Config page, enter the IP address, location, Link Type or Status and click on the filter button to search the IP address previously added.
  • To clear the current search criteria, search results and enter new criteria click on the clear button.
  • Click on the Edit button against the IP address that has to be modified.

  • Edit IP Config page has similar fields and functions like Add IP Config page.
  • Make desired changes in the IP Address or Location
  • Click on the Update button to save the changes and close the pop-up window or click on Cancel button to exit the pop-up window without saving changes.
