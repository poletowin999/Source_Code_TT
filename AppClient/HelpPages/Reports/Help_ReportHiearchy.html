User Hierarchy Report

REPORT DESCRIPTION

The user hierarchy report displays list of employees who are attached to selected supervisor.

ACCESSIBLE BY

End Users: Can view the user information and his/her supervisor

Supervisor or Manager: Can view the subordinates attached to them

Administrator: Can view all employees and their subordinates

STEPS TO ACCESS THE REPORT

  • From the Home screen click on Reports
  • From the Reports screen click on User Hiearchy Report

STEPS TO RUN THE REPORT

  • Select the Location
  • Then select the Supervisor in the selected location
  • Click on the View button to view the list of subordinates attached to the selected supervisor
  • To exit without running the report click the Cancel button

OUTPUT

The below fields are displayed in the report

Supervisor Name, User Name, Role Name, Email ID, Contract Type and Location

Report Options

  • The left arrow and right arrow button helps in navigating to the previous page and next page of the report.
  • The forward and backward button helps to navigate to last page and first page of the report.
  • We can also search desired value by entering the value in the search box and click on Find button. Click Next to search the value again.
  • Click on the Save icon, which provides option to export the report in Excel, Word or PDF.
