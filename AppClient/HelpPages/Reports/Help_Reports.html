Reports

The following are the list of reports that are available. All reports will run for a user and subordinates attached to him / her (if any). Administrator can view reports for all users across all locations.

ACTIVITY ENTRY REPORT

The activity entry report displays the details of activities entered the logged in user and subordinates attached to him / her

ACTIVITY NOT ENTERED REPORT

The activity not entered report display the list of subordinates attached to the logged in user and dates the subordinates were present and didn’t enter any activity. User is considered as present on a day if the user has entered the check in time.

BILLING REPORT

The billing report gives total working hours spent for work type and billing type under a project for each client. The report also includes the miscellaneous activities entered for the project.

PAYROLL REPORT

The Payroll report gives user wise total working hours, including the hours spent on miscellaneous activities.

EMPLOYEE CHECK IN/OUT DETAILS

The Employee Check In/Out details gives the check in and check out timings of the logged in user and subordinates attached to him / her.

USER HIERARCHY

The User Hierarchy report shows the employee structure attached to the selected supervisor and location.

UNAPPROVED ACTIVITIES

The Unapproved Activities report shows the list of activities that has ot be approved by the selected supervisor.

USER MASTER REPORT

The user master report displays users details like, Employee ID, Employee Name, Role, etc.

CHECK IN/OUT MISMATCH WITH ACTIVITY DURATION REPORT

Report displays list of user wise activity duration if the total activity duration entered by the user does not match with the Check In and check Out duration
