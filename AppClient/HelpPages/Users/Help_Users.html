User Management

FUNCTION

  • The Users page helps us to add a new user or update / change details of an existing user.

ACCESSIBLE BY

  • Users who have administrator access.

STEPS TO CREATE A NEW USER

  • From the Home page, Admin  Users
  • This will take us to the Users Management page.

  • Click on the Add New button to open the Add user page
  • Enter the employee’s Last name, First Name and Initial
  • Select the Gender from the drop down menu
  • Enter the Home phone number of the employee
  • Enter the Employee ID of the employee
  • Select the User type (e.g. Permanent, Contract, Casual, etc.,) from the drop down menu
  • Enter the Role or press F2 key to select a Role from the drop down
  • Enter the Office phone number of the employee
  • Enter Extension to reach the employee in the office
  • Enter the Location or press F2 key to select a Location from the drop down
  • Check mark the Is Admin User check box to provide administrator rights to the employee. This will provide access to all pages and all data for that particular user / employee. Also then admin user can view everyone’s data regardless for location
  • If the Activities Auto Approval check box is checked, the activities entered doesn’t require an approval from the employee’s superior. The activities entered will get approved automatically
  • To allow users to access the application from outside the office network mark the "Allow user from anylocation" check box
  • Enter the Date of joining
  • Enter the Email ID. The email id will be helpful to reset forgotten passwords
  • Enter the Login name or user name which will be used by the employee to access the application.
  • Enter the password in Password and Confirm password fields
  • To enter the name of the manager. The user will be attached to the selected supervisor, so that we need not go to user hierarchy screen to add the superior for this user
    • Press F2 or click on the icon to open the search window
    • Select the City or enter Name or Role or Email ID of the manager to search.
    • Click on the Filter button to display employees meeting the above search criteria.
    • Click on the Close button to exit without saving.
    • To select a manager click on the Select button or click on the Clear button to clear the search criteria and results.

  • Click on the Add button to save the information and create the user
  • Click on the cancel button will clear the entered information and takes back to the users management page

Note: Fields that are marked with an asterix (*) are mandatory fields

STEPS TO EDIT AN EMPLOYEE INFORMATION

  • To change or update any employee’s information, search the employee from the Users management page using any one of the following data or combination of these.
    • Employee Name, Role Name, Email ID, Location or status
  • Type few letters of one or more fields and click on the filter button to search the employee matching the above criteria.
  • To clear the current search criteria, search results and enter new criteria click on the clear button
  • To edit an employee details, click on the Edit button against the particular employee, which will take us to the Edit User page to change or update the employee information.

  • The Edit user page will display all the fields and information that were entered during adding an employee or user.
  • In addition to this the Edit user page has option to activate or inactivate an employee. To inactivate an employee uncheck the Active check box.
  • When a user with subordinates is inactivated, the users attached to the inactived user will automatically move to the superior of the inactivated user. The inactivated user will be removed from the hierarchical structure
  • The Edit User page also has option to enter the Relieving Date if an employee has quit the organization
  • Update or change the desired information / field.
  • Enter the reason for making changes in the employee information.
  • Click on the Update button to save the changes and exit or click on the Cancel button to exit without saving any changes to the employee data
  • Users with Administrator access can reset other employee’s password from this page.
  • When the user edits the data, the password will not be populated. So user need not enter the same password to update. Enter a password only when the password needs to be changed or reset, if not leave it blank to retain the existing password.
