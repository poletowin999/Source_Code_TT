Payroll Report

REPORT DESCRIPTION

The report displays user wise total work duration during the specified date range. Only approved activity’s duration will be taken into account for calculating the total duration. Report can be used for project wise invoicing purpose

ACCESSIBLE BY

End user: Can view total duration of activities for the given period for the logged in user

Supervisor or Manager: Can view employee wise total duration of activities for the given period for the logged in user and subordinate’s attached to him / her

Administrator: Can view employee wise total duration of activities for the given period for all users irrespective of location

STEPS TO ACCESS THE REPORT

  • From the Home screen click on Reports
  • From the Reports screen click on Payroll Report

INPUT PARAMETERS

  • Enter the From Date and To Date, within which we wish to see the month wise work duration.
  • The report can be filtered based on the below fields. Select the fields from the drop down, multiple selections is also possible. If no field is selected report will run for all fields.
    • Location
    • Users
    • Projects
    • Language
  • Click on View button to run the report
  • To exit without running the report click the Cancel button

OUTPUT

  • After inputting the desired conditions, click on the View Report button to view the summary report
  • The summary report will display the total duration spent by each employee in a month against a project.

Output Fields

Following fields are displayed in the report

  • Employee name, Location, contract type, Role name, Client Name, Project Name, Bill type, work type, Language, Activity type, Hours, Minutes, Total Hours.

REPORT OPTIONS

  • The left arrow and right arrow button helps in navigating to the previous page and next page of the report.
  • The forward and backward button helps to navigate to last page and first page of the report.
  • We can also search desired value by entering the value in the search box and click on Find button. Click Next to search the value again.
  • Click on the Save icon, which provides option to export the report in Excel, Word or PDF.
