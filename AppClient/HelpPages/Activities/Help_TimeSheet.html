Time Sheet

FUNCTION

  • The Time sheet page enables user to capture the check in time and check out time for the current day. We have employees working in different time zones and cross shifts, to allow the users to capture the check In and Check Out time, the page allows users to capture the previous day and the subsequent day timings.
  • The Time Sheet page also helps user to view the check in time and check out time for each and every day.

ACCESSIBLE BY

  • All Details.

  • From the Home page, My ESS  Time Sheet
  • This will take us to the User Swipe View page.

  • Click on the Check In to update the swipe in timings
  • Then select the Time Zone in which the user is working.
  • Then enter the shift in which the user is working
  • Click OK button to capture the current timing as the Check In time.
  • Similarly we can capture the Check out Time.
  • The check in time and check out time are captured based on the time zone in which the employees work.
  • User can we previous month and next calendar by pressing the Previous / Next link on top of the calendar

Note: User cannot update or change the Check In or Check Out time, the application captures the current timing based on the time zone in which the employee works, so this activity has to be done during the beginning of the shift to capture the check in time and end of the shift to capture the check out time.
