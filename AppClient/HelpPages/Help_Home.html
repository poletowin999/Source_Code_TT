HOME PAGE

The Page displays the Working duration and activity status in a bar graph format for the user who logins in

Work Duration Graph

  • The work duration graph displays the duration for each day based on the check in and check out time.
  • Work duration is displayed for past one month from current date.
  • Duration (in Hrs) is displayed on the Y-axis and dates on the X-axis.

Activity Status

  • The Activity Status graph displays the number of activities that are
    • Waiting for approval,
    • Rejected activities
    • Approved activities
  • Activity status is displayed for past one month from current date.
  • Number of activities is displayed on the Y-axis and dates on the X-axis.
